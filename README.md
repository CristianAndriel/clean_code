# Clean Code
O termo Clean Code é o título do livro de Robert C. Martin onde ele descreve os princípios que ele "entende" devem ser seguidos para se manter um código limpo, de qualidade e que seja fácil de ser mantido.
Neste repositório, deixo minhas próprias anotações, exemplos de código e afins, estudados a partir do conteúdo compartilhado no Clean Code.
